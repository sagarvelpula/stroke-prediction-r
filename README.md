# Stroke Prediction using R (Machine Learning)

This project predicts the likelihood of stroke using healthcare data and logistic regression.

## Overview

The project includes:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Logistic Regression model
* Model evaluation using accuracy and ROC curve

## Features

* Missing value handling
* Outlier treatment
* Data visualization using ggplot2
* Predictive modeling
* ROC & AUC evaluation

## Technologies Used

* R Programming
* dplyr, ggplot2
* caret
* pROC
